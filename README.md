# 中小学STEAM教育大会

## 简介

一年一度，坚持公益。从2013年的STEAM教育创新论坛到2016年的STEAM教育大会，这个最早由民间一线教师发起的创客教育、STEAM教育方面的教育活动，逐渐发展成为国内综合学科方面最有影响力的教育活动，引领中小学创客教育和STEAM教育的前沿发展。同时，大会陪同国内创客、STEAM、人工智能、跨学科创新教育等领域的骨干教师一路成长，成为他们最美好的记忆之一。
